# Douyin Codex Main Image Demo Design

## Style Prompt

A fast vertical Douyin/TikTok tutorial promo for a Codex ecommerce plugin. The visual language is dark, technical, and high-conversion: real screen recording as proof, oversized Chinese hooks, neon cyan and ecommerce orange highlights, compact product-result cards, progress energy, and clear comment CTA. The video should feel like a useful AI workflow discovery, not a software documentation walkthrough.

## Colors

- Background: `#080A10`
- Panel: `#111722`
- Panel border: `#263345`
- Primary text: `#F6FAFF`
- Muted text: `#A7B2C2`
- Codex cyan: `#20E7D6`
- Commerce orange: `#FF9F1C`
- Alert red: `#FF4D6D`

## Typography

- Main Chinese UI/font target: Arial Unicode MS or available system CJK fallback
- Headline style: bold, 78-92px, tight but readable, no negative letter spacing
- Supporting captions: 34-44px
- Labels: 28-32px

## Motion Rules

- Keep cuts fast: 2-7 second beats.
- Use the screen recording as continuous proof, but compress waiting with captions.
- Add large text overlays that change every few seconds.
- Use highlight boxes and tags instead of long paragraphs.
- End with a simple comment CTA.

## What NOT to Do

- Do not make it look like a static tutorial screenshot.
- Do not over-explain the plugin architecture.
- Do not cover the product images with large text.
- Do not use low-contrast gray text.
- Do not leave the raw screen recording full-frame without packaging.
